# Instalação
- via Pypi
```commandline
pip install MspeechPy
```
- Direto pelo Git
```commandline
pip install git+
```

# dependências:
- caso necessite instale manualmente 
```commandline
pip install azure-identity azure-spe
```

# instale o azure-developer-cli para seu sistema:
